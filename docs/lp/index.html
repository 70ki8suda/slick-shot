FAQ

よくある質問

対応OSは？

macOS 専用です。Windows では利用できません。公開時には対応OSの下限を明記してください。

画面収録の許可は必要？

はい。初回利用時に macOS の画面収録権限が必要です。

アップデートは含まれる？

SlickShot は買い切りです。少なくとも同一メジャー内のアップデートを含め、Sparkle ベースのアプリ内自動更新に対応できます。
